Historic Sites is a collaborative capstone project for Deep Dive Coding, a ten week intensive web development bootcamp through CNM Ingenuity in Albuquerque, NM. The site is built with React using JSX, Babel, Redux and Webpack. The backend is handled by Node.js and Express and uses a MySQL database. The goal of the site is to create a dataset for historic sites in the United States (starting with New Mexico), and to allow users to easily search for sites that might interest them. Search results are based off location and specific tags for historically unique periods. Registered users can add sites to a travel list and upload new historic sites to the database.

Liquid Barn is an online seller of DIY eLiquid products based out of Simi Valley, CA. Their website is built on the ecommerce platform Shopify, using the Pipeline theme from Groupthought. A significant amount of custom CSS has been added to the theme, as well as numerous landing pages built entirely with custom HTML, CSS and JavaScript, including the DIY Kits page and Tastemaker Collection brand pages. Custom JavaScript and Python scripts have also been integrated into backend processes to help manage inventory and restrict shipping options for specific products/variants. Product images, graphics and video also created by me.

This calculator was built for Liquid Barn to help customers more easily and accurately calculate measurements when mixing their own eLiquid. The calculator is built entirely with HTML, CSS and jQuery. It calculates on every change in input and displays the results so that customers can quickly adjust their values and get immediate feedback. Default values can be selected and calculations updated based on the measurement preferences of the customer. Flavors can be indefinitely added and removed. The calculator also makes use of the jsPDF library so that customers can download an exported pdf version of the recipes they create.

This simple game is based off the Simon Electronic Memory Game, which displays a random pattern that the player must memorize and repeat back in order to progress to the next level. Each new level adds to the preceding pattern, making it longer and more difficult to remember. Built to further my understanding of programming and jQuery specifically, it pushes a random number into a stored array at every level and then checks the array against the values inputted by the player. If they match, the game continues, otherwise the screen flashes game over and the game resets. Give it a go and see how far you can make it.
